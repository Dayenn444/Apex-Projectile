# Panduan Kontribusi (Contribution Guide)

Terima kasih telah ikut berkontribusi dalam proyek ini! Untuk menjaga kerapian kode dan riwayat komit (*git history*), silakan ikuti panduan langkah demi langkah di bawah ini.

---

## 🛠️ Alur Kerja Kontribusi

### 1. Kloning (Clone) Repository
Langkah pertama adalah menyalin repository ini ke komputer lokal Anda:
```bash
git clone [https://github.com/USERNAME/NAMA_REPOSITORY.git](https://github.com/USERNAME/NAMA_REPOSITORY.git)
cd NAMA_REPOSITORY
```

# Pastikan Anda berada di branch main yang paling update
git checkout main
git pull origin main

# Buat dan pindah ke branch baru (gunakan nama yang deskriptif)
git checkout -b fitur/nama-fitur-anda
# Contoh: git checkout -b perbaikan/issue-12
